What are the different types of selectors?

Selector Name
Selector Example
  • Type selectors -- matches elements by node name
  • Class selectors -- matches elements by class name
  • ID selectors -- matches elements by ID name
  • Universal selectors -- matches elements of any type
  • Attribute selectors -- matches elements based on the presence
    • or value of a given attribute (e.g. a[title] matches all a elements with a title attribute)
  • Descendant selectors -- matches elements where element B is a descendant of element A
  • Direct child selectors -- matches elements where element B is a DIRECT descendant of element A
  • Adjacent sibling selectors -- matches elements where element B follows right after A
  • Pseudo-Class -- attaches to a css selector to specify a special state
  • Pseudo-Selector -- attaches to a css selector to specify a oseudo-element as a child of the property selected

  • div > p (would select every p tag that is a DIRECT descendant of a div tag)

  • div:hover (when hovered, do the following things....)
  • div::before (before the selected div, do the following things....)
